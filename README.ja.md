# AE-SaveGuard
## 作業に集中してください。保存は AE Save Guard が守ります。



# こんな方におすすめ

- [ ] 保存し忘れて作業を失ったことがある
- [ ] 自動保存を信じていたのにデータが消えた
- [ ] 何度も保存しても不安になる


# 機能

- 起動するだけで、保存タイミングをやさしく通知
- 複雑な設定は不要

### 1. インストールするだけ

- AE起動時に自動実行
- 手動起動不要
- 再起動後も維持

### 2. 設定なしで使用可能

- プロジェクトを自動検出
- パス設定不要

### 3. 作業を邪魔しない

- 必要な時だけ通知
- 集中を妨げない

### 4. カスタマイズ可能

- 通知画像変更
- 通知音変更


# インストール

## 1. ダウンロード

- https://github.com/srejis/AE-SaveGuard/releases

## 2. インストーラー実行

- `AESaveGuard-Setup.exe` 実行
- 「起動」にチェック → 完了


# 使用方法

## ⚙️ 設定（任意）

- 初回起動時に表示
- 設定後「Apply」をクリック

### 再度開く

- トレイアイコン → 右クリック → 設定


# アンインストール

- Windows設定 → アプリ → 削除  
または  
- コントロールパネル → アンインストール
