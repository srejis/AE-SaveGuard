# AE-SaveGuard
## **작업에만 집중하세요, 저장은 AE Save Guard가 챙깁니다**

# **이런 분들께 추천합니다**

- [ ]  **저장 안 누른 채 작업하다가 날려본 적 있는 분**
- [ ]  **자동 저장 믿었다가 작업 잃어본 적 있는 분**
- [ ]  **저장 습관은 있지만, 불안해서 한 번 더 확인하는 분**


# 기능

- **그냥 켜두면, 저장 타이밍만 가볍게 알려줍니다**
- 복잡한 설정 없이 바로 사용할 수 있습니다

### 1. 설치하면 끝입니다

- AE를 켜면 자동으로 함께 실행됩니다
- 따로 켤 필요 없습니다
- 컴퓨터를 껐다 켜도 그대로 유지됩니다

### 2. 아무 설정 없이 바로 사용 가능합니다

- 작업 중인 프로젝트를 자동으로 인식합니다
- 저장 경로나 파일 위치를 따로 설정할 필요 없습니다

### 3. 작업 흐름을 방해하지 않습니다

- 필요한 순간에만 가볍게 알려줍니다
- 작업 중 몰입을 깨지 않습니다

### 4. 필요하면 원하는 방식으로 바꿀 수 있습니다

- 알림 이미지 변경 가능
- 알림 소리 변경 가능

# 설치 방법

## 1. 다운로드

- 아래 링크에서 최신 버전을 다운로드합니다
- https://github.com/srejis/AE-SaveGuard/releases

## 2. 설치 파일 실행

- `AESaveGuard-Setup.exe` 파일을 실행합니다
- 설치 완료 창에서 **실행 체크 후 Finish**를 눌러주세요

# 사용 방법

## ⚙️ 설정 (선택)

- 처음 실행 시 설정 창이 한 번 표시됩니다
- 원하는 설정 후 **Apply 버튼**을 누르면 적용됩니다

### 설정 다시 열기

- 작업표시줄 오른쪽 (트레이 아이콘) → 우클릭
- **Setting 열기** 선택

# 삭제 방법

- Windows 설정 → 앱 → **AE Save Guard 제거**
    
    또는
    
- 제어판 → 프로그램 제거 → 삭제
