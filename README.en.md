# AE-SaveGuard
## Focus on your work. AE Save Guard takes care of saving.



# Who is this for?

- [ ] You’ve lost work because you forgot to save
- [ ] You trusted auto-save and still lost your project
- [ ] You save often, but still feel uneasy


# Features

- Just run it — it gently reminds you when to save
- No complicated setup required

### 1. Install and you're done

- Runs automatically when AE starts
- No need to launch manually
- Works after reboot

### 2. Works out of the box

- Automatically detects your project
- No need to configure paths

### 3. No interruption

- Only notifies when necessary
- Doesn’t break your focus

### 4. Customizable

- Change notification image
- Change notification sound


# Installation

## 1. Download

- https://github.com/srejis/AE-SaveGuard/releases

## 2. Run installer

- Run `AESaveGuard-Setup.exe`
- Check **Launch after install** and click Finish


# Usage

## ⚙️ Settings (Optional)

- Appears once on first launch
- Click **Apply**

### Open settings again

- System tray → Right click → Open Settings


# Uninstall

- Windows Settings → Apps → Remove  
or  
- Control Panel → Uninstall program
